# Esta pasta contém snippets para o VSCode

## Como instalar no VSCode

### Opção 1 (recomendada para time)

1. Abrir:
    → `File → Preferences → User Snippets`

2. Criar:
    → `<nome_snippet>.code-snippets`

3. Colar o JSON

### Opção 2 (via repo do playbook)

1. Colocar em:
    → `.vscode/<nome_snippet>.code-snippets`
