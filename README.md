# AI Dev Playbook

Guia oficial de uso de IA no desenvolvimento.

## Objetivo
Padronizar uso de IA para:
- aumentar produtividade
- reduzir erros
- melhorar troubleshooting

## Como usar (rápido)

1. Gere contexto do projeto:
   → use `context/ai-context-template.md`

2. Planeje:
   → use `templates/planning.md`

3. Refine:
   → use `templates/refine.md`

4. Execute:
   → use `templates/execute.md`

## No VSCode

Importe os snippets em:
snippets/vscode.code-snippets

## Regra principal

Nunca usar IA sem contexto + validação
